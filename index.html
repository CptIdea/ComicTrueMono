Tables that actually line up.

      and junction softly rounded in the Comic style, and woven double joints. Mixed
      single/double borders with different column widths tile seamlessly, so real tables just work.

How it was made

    
Assembled from unmerged pull requests.

    
Good contributions to the free monospaced Comic Sans forks sat unmerged for years —
      the original Comic Shanns went quiet in 2023, and even the maintained repos still ignore font PRs
      (kaBeech's italics have been open since 2024). Comic True Mono gathers that scattered work into one
      place — every glyph from an MIT source — and constructs the few that existed nowhere.

    
Comic MonoMIT
dtinth
The monospaced base — clean 1116-unit metric.
ABCDEFGHIJKLMNOPQRSTUVWXYZ a b c 0 1 2
Comic ShannsMIT
Shannon Miwa
The root letterforms + full European Latin.
À Ç É Ñ Ö Ü ā ć đ ģ ķ ł ń ő ř š ž
Serious ShannsMIT
kaBeech
The λ glyph, hand-drawn italics, and Ł ł.
λ Ł ł  f g k x  (italic)
clsn forkMIT
clsn — PR open since 2021
Ligatures & rare letters nobody merged.
Æ æ Œ œ Þ þ ſ ƿ Ƿ ꝛ ∀ ∃ ∄
lilmayu forkMIT
lilmayu — PR open since 2022
Correctly rotated caron / háček.
Č č Ě ě Ř ř Š š Ž ž ˇ
In-projectnew
constructed here
Absent from every source — built by hand.

Get it

    
Install or build.

    

      

        Install
        
Download the family and install (double-click on macOS/Windows, or drop into
          ~/.local/share/fonts on Linux). Set your editor font to Comic True Mono.

        Browse fonts/ttf ↗
      

      

        Build from source
        
Requires FontForge + fontTools. Everything derives from one master.

        
pip install -r requirements.txt
make            # build fonts + specimens
